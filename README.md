Project Overview

This project is a simple web application where users can search for different mobile phones and view their details. The main idea was to build something that looks clean and is easy to use, while also working with real data from an API.

🔹 How I Built This Project

To complete this project, I worked step by step on both the design and the functionality.

I started by creating the basic layout using HTML and styled it with Tailwind CSS so that the website looks modern and responsive. After that, I connected the project to the Phone Hunting API to fetch phone data dynamically based on what the user searches.

I added a search feature where users can type the name of a phone and get results instantly. The phones are displayed as cards, and when a user clicks on “Show Details”, a popup (modal) appears showing more information about that phone.

I also implemented a “Show All” button so users can view all the results instead of just a few. Since the API doesn’t provide price details, I added a separate catalog section at the bottom with manually added prices to match the design shown in the reference.

🔹 Features
Search phones by name
Display results dynamically in card format
View detailed phone information in a modal
“Show All” functionality for more results
Additional catalog section with phone prices
Clean and responsive UI
🔹 JavaScript Concepts Used

While working on this project, I used:

Fetch API to get data from the external API
Async/Await to handle API responses
DOM manipulation to display data dynamically
Event handling for buttons and user actions
Array methods like slice() and forEach()
Template literals to insert dynamic content
🔹 Challenges Faced

One issue I faced was that the API didn’t provide phone prices. To solve this, I created a separate section where I manually added phone prices so that the website looks complete and closer to the original design.
